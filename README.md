# MdxAcademy - Product Page

MdxAcademy is a product page whereby users can enrolled into particular lessons.

---

#### Features

- **Add to cart** - User can add particular lesson to shopping cart.
- **Full text search** - User can search for any particular lesson across different categories.
- **Filters** - User can filter by any category by ascending or descending order.
- **Checkout** - User can purchase lessons they are enrolled into.
- **Database** - MongoDB Atlas has been used to store lessons info and orders purchased.
- **Frontend** - Responsive UI entirely build using HTML, CSS and Vue 2.x.x.
- **Backend API** - Node and Express JS have been used.

#### Tech Stack

- Frontend: HTML, CSS and Vue 2.
- Database: MongoDB Atlas.
- Hosting: GitHub Pages (Frontend) + Render (Backend).
